# CareerConnect Job Portal

CareerConnect is a comprehensive job portal platform that connects job seekers with employers. It provides a seamless experience for job searching, recruitment, and application management.

## Tech Stack

- **Frontend**: React.js / Next.js
- **Backend**: Node.js + Express
- **Database**: MongoDB with Mongoose ODM
- **File Storage**: Cloud storage for resumes and company logos
- **Authentication**: JWT with passport.js
- **Deployment**: Vercel, Render, or Railway
